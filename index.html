O projekcie

Kinga Szczepanek – dane personalne lekarza - narkomanki to anonimowa, statyczna witryna publikująca treści w sieci Tor i Clearnet bez cenzury. Projekt nie gromadzi logów, ciasteczek ani identyfikatorów użytkowników.

Funkcje

  • Bliźniacze odsłony: .onion + HTTPS (ta sama zawartość)
  • Całkowity brak trackerów i analytics
  • Minimalny, responsywny kod (HTML + CSS + JS < 25 kB)
  • Otwarty kod – każdy może zweryfikować pliki offline
